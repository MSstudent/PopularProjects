#.NET 优秀开源框架列表

*这里收集了一些.NET平台的优秀开源框架，方便大家借鉴、使用。*

##.NET

标题 | 地址 | 说明
------------ | ------------- | ------------
xUnit.net | http://xunit.codeplex.com/ | 开源、免费的单元测试项目
RhinoMocks mocking framework | http://ayende.com/blog | 开源的测试项目，通过创建mock使测试更简单
White for automation of Windows applications | http://teststack.github.io/White/ | 代码驱动Windows程序测试
Gallio Automation Platform  | http://www.gallio.org/ | 运行很多测试框架，如MSTest、Xunit、MbUnit
Fluent Nhibernate | http://www.fluentnhibernate.org/ | 让你可以用C#代码来设置数据映射关系，ORM开源框架
StructureMap Dependency Injection/Inversion of Control  | http://docs.structuremap.net/ | 项目的解耦和依赖框架，可以提高项目和代码质量
Managed Extensibility Framework | http://mef.codeplex.com/ | 从静态编译程序转换到动态语言程序
sharp-architecture | https://code.google.com/p/sharp-architecture/ | 用ASP.NET MVC和NHibernate快速开发web应用程序
OpenRasta REST based framework for building web applications  | http://openrasta.org/ | 让你的程序拥有一个REST API接口
CSLA.NET Application Framework | http://www.lhotka.net/cslanet/ | .NET开发综合框架
Spring.NET Application Framework  | http://www.springframework.net/ | Web开发综合框架，让你的应用程序使用Spring
Mono enables .NET on Linux and Mac | http://www.mono-project.com/Main_Page | 在Linux、BSD和OS X上使用.NET
Sandcastle Help File Builder | http://shfb.codeplex.com/ | 创建MSDN样式的文档
EasyHook for Windows API Hooking  | http://easyhook.codeplex.com/ | 用托管代码扩展非托管代码
Json.NET for working with JSON formatted data | http://json.codeplex.com/ | 使用.NET对JSON数据进行处理
Excel Data Reader for Excel 97 to 2007 | http://exceldatareader.codeplex.com/ | 将Excel文件读取到Dataset中
C# Based Open Source SNMP for .NET and Mono | http://sharpsnmplib.codeplex.com/ | 对SNMP的包装
DotNetZip Library | http://dotnetzip.codeplex.com/ |  ZIP库处理和示例
VisioAutomation | http://visioautomation.codeplex.com/ | .NET语言对Visio进行处理
Argotic Syndication Framework for RSS, Atom, OPML and more | http://argotic.codeplex.com/ | 读写聚合内容的库
NLog logging library | http://nlog-project.org/ | 调试程序库

##WP

标题 | 地址 | 说明
------------ | ------------- | ------------
Open Data Protocol | http://odata.codeplex.com/ | WP 定义数据交换的协议，用来数据查询和更新的web框架
Microsoft Silverlight Analytics Framework | http://msaf.codeplex.com/ | 可扩展的Web分析框架
Silverlight for Windows Phone Toolkit | http://silverlight.codeplex.com/releases/view/55034 | WP第三方框架，增加控件、行为等内容
Microsoft Silverlight Media Framework | http://playerframework.codeplex.com/ | 开源强大的视频播放框架，支持WP、Win8、WPF等
Silverlight SharpZiplib | http://slsharpziplib.codeplex.com/ | 从C# library SharpZiplib移植而来的Silverlight压缩库，支持Windows Phone 
UnitDriven | http://unitdriven.codeplex.com/ | 这是一个跨平台的CLR测试框架。它允许你编写一个可以在Silverlight,，Windows Phone 7和其他.NET平台中同步或异步运行的独立测试
HugeFlow MVVM Library | http://hugeflowmvvm.codeplex.com/ | 这个框架可以使你更轻松地在Silverlight或Windows Phone 7应用程序中运用MVVM模式，同时它可以与Expression Blend进行交互
CodeTitans | http://codetitans.codeplex.com/ | CodeTitans代表了一系列的类库，包括JSON@CodeTitans，它可以使你更轻松地使用JSON库，还包括IoC@CodeTitans，一个很有特点的反转控制的实现，这个类库可以用于.NET 2.0+，.NET CF 2.0以及Mono 2.0+，当然还有Silverlight和Windows Phone 7
MVVM Light Toolkit | http://mvvmlight.codeplex.com/ | 这是一个支持Windows Phone 7/Silverlight/WPF的MVVM框架，由微软的MVP Laurent Bugnion提供，此框架还提供了Visual Studio和Expression Blend的模板及智能感知支持
Daenet | http://mobilesyndication.codeplex.com/ | 这是.NET Compact Framework和Windows Phone 7平台上第一个支持RSS 2.0 和Atom 1.0聚合反馈的类库
Windows Phone 7 Multi-Touch Manipulation | http://multitouch.codeplex.com/ | 该项目附带了一个实用的多点触控库，并实现了惯性效果和单手转动，即便在Windows Phone 7模拟器上也可以模仿实际设备上的多点触控效果（前提是你的电脑能识别多点触摸）
Independ | http://independ.codeplex.com/ | Independ是一个Windows Phone 7平台上的轻量级依赖注入容器
ESRI ArcGIS Silverlight Toolkit | http://esrisilverlight.codeplex.com/ | 这个项目是ArcGIS API的Silverlight/WPF扩展，目前已经支持Windows Phone 7。它包含了大量控件，小部件和工具
WriteableBitmapEx | http://writeablebitmapex.codeplex.com/ | 这是Silverlight中WriteableBitmap类的一系列扩展方法，支持Windows Phone 7平台。
Windows Phone 7 Accelerometer Simulator Kit | http://wp7accelerometerkit.codeplex.com/ | 这是一个可以在Windows Phone 7模拟器中体验重力感应器的类库
WP7 GPS Simulator | http://wp7gps.codeplex.com/ | 通过使用这个项目我们可以在Windows Phone 7模拟器或设备中模拟GPS数据
Bewise Phone Controls | http://bewisephonecontrols.codeplex.com/ | 为Windows Phone 7开发人员提供了大量的控件，组件和工具，它同时支持Silverlight for Windows Phone和XNA平台。
Quebec | http://quebec.codeplex.com/releases/view/54885 | 这个项目包含了很多Silverlight，WPF和Windows Phone 7平台的扩展工具
Coding4Fun | http://coding4fun.codeplex.com/ | 包含了一些控件等
